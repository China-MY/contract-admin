## 六、核心交互特性

### 6.1 SelectCreate 组件（`src/components/SelectCreate.vue`）

通用可创建下拉选择器：
- **搜索** — 输入文字自动过滤选项
- **创建** — 输入不存在内容时，下拉出现「➕ 创建 xxx」选项，点击创建
- **自动匹配** — 输入内容与已有选项精确匹配时自动选中

### 6.2 已应用 SelectCreate 的位置

| 页面 | 字段 | 创建类型 |
|------|------|---------|
| 合同表单 | 我方公司 | 直接写入值 |
| 合同表单 | 对方单位 | POST /api/customers |
| 合同表单 | 关联项目 | POST /api/projects |
| 发票表单 | 关联合同 | POST /api/contracts |
| 发票表单 | 发票类型 | 直接写入值 |
| 发票表单 | 开票方 | 直接写入值（选项=公司列表） |
| 发票表单 | 收票方 | 直接写入值（选项=客户列表） |
| 资金计划/记录 | 关联合同 | POST /api/contracts |
| 资金计划/记录 | 付款方/收款方 | 直接写入值（选项=客户+供应商） |
| 项目表单 | 客户名称 | 直接写入值（选项=客户列表） |
